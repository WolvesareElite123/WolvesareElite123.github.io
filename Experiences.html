Web Application

Status: Complete


About the Project | Motivation

The Medical Literacy and Access Project was created by a group of college students from several different universities. The project was inspired by our first-generation parents struggles of finding medical care within the United States after moving here as immigrants.

The goal of our project is to create an immigrant-friendly website that assists them in finding a compatible medical professional near them. The project utilizes translation tools to convert the website to a language of the users choosing then prompts them with filters to find the most ideal doctor near them

To accomplish the filtering and to have a database of doctors to use, we have partnered with USA News to access their pre-existing database


Technical Work

  • Created the algorithms necessary to read US News query parameters
  • Created functions to create our own strings using imported parameters
  • Incorporated Google's Language Translation API
  • Designed and created an accessible web-form
